# 🧾 Claim & Refund Journal System  
Health Insurance

---

## 📌 Overview
This project focuses on developing a claim and refund journal system for a health insurance provider to manage claim disbursement, refund transactions, and financial journal entries.

Previously, claim and refund processes were handled manually, resulting in delays, data inconsistencies, and difficulties in tracking financial transactions and reconciliation.

The system is designed to streamline claim processing, ensure accurate journal recording, and improve financial transparency and control.

---

## 🎯 Key Features
- Claim Transaction Management  
- Refund Processing  
- Journal Entry Recording  
- Claim Status Tracking  
- Financial Reporting & Reconciliation
